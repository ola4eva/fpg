Dynamic Purchase Approval.

Key Features:

  • Allows you to approve purchase order by users or security group wise.
  • User can configured purchase approval dynamically.
  • User can configure purchase approval upto any level.
  • Support With Multi Company and Multi Currency.
See Video

Overview

  • From comapny form view, user can configured the purchase approval process as per below image.
  • User can configured from amount, to amount, approve process and approval level as per below image.
  • When user confirm purchase order if any approval configured between purchase order total amount then it goes to the "To Approve" state.
  • Whoever user have approval rights can approve the purchase order as per below image.
  • Once all approval levels are done then purchase order moved to confirm state and it works as per odoo default flow.
